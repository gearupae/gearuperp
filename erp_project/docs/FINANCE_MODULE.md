## üìã Features

### 1. Chart of Accounts
**URL**: `/finance/accounts/`

| Feature | Description |
|---------|-------------|
| Hierarchical Structure | Parent-child account relationships |
| Account Types | Asset, Liability, Equity, Income, Expense |

## üîÑ Posting Flows (SAP/Oracle Standard)

### Sales Invoice Flow
```
Draft ‚Üí Posted (Dr AR, Cr Revenue, Cr VAT) ‚Üí Payment Received (Dr Bank, Cr AR)
```

### Vendor Bill Flow
```
Draft ‚Üí Posted (Dr Expense, Dr VAT, Cr AP) ‚Üí Payment Made (Dr AP, Cr Bank)
```

### Expense Claim Flow
```
Draft ‚Üí Submitted ‚Üí Approved (Dr Expense, Cr Employee Payable) ‚Üí Paid (Dr Employee Payable, Cr Bank)
```

### Payroll Flow
```
Draft ‚Üí Processed (Dr Salary Expense, Cr Salary Payable) ‚Üí Paid (Dr Salary Payable, Cr Bank)
```
